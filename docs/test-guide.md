## 测试前置准备

### 启动项目

```bash
npm install
npm run dev
# 浏览器打开 http://localhost:5173
```

### 进入调试模式

在开始界面"道号"输入框输入 **`Debug`**（区分大小写），点击"开始修炼"。  
界面底部会出现 **🐛 调试** 折叠面板，展开后可直接修改数值和添加物品。

### 常用调试操作速查

| 操作 | 方法 |
|------|------|
| 修改任意属性 | 调试面板 → 数值 Tab → 修改对应字段 |
| 一键满状态 | 调试面板 → 数值 Tab → 「♻️ 全部恢复」按钮 |
| 添加物品 | 调试面板 → 物品 Tab → 选择物品 → 输入数量 → 添加 |
| 添加装备 | 调试面板 → 物品 Tab → 装备区域 → 添加 |
| 快速升级 | 调试面板 → 数值 Tab → 修改 `exp` 到突破阈值，或直接修改 `realmIndex` |
| 添加灵石 | 调试面板 → 数值 Tab → 修改 `gold` 字段 |
| 重置存档 | 浏览器 DevTools → Application → Local Storage → 清除 |
