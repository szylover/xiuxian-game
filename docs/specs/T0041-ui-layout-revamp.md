# 设计文档：界面布局改版（三栏布局 + 头像系统）

任务：T0041
日期：2026-03-27

## 概述

当前界面为单栏垂直布局（720px 最大宽度），所有面板按顺序堆叠：状态栏 → 展开面板（状态/背包/炼丹/装备/商店/炼器） → 操作按钮 → 日志。面板多时需要频繁展开/收起，操作效率低且视觉体验拥挤。

本次改版将界面重构为 **三栏布局**：

- **左栏（固定）**：角色头像 + 核心属性，提供"详细"按钮展开完整属性面板
- **中栏（主区域）**：操作按钮 + 游戏日志
- **右栏（可切换）**：功能面板按钮组（分组排列），点击后在右栏展开对应面板内容

同时引入 **头像系统**，在左栏角色面板顶部醒目展示。

### 关键设计决策

1. **单面板展开**：右栏同一时间只展开一个面板（类似标签页），避免右栏无限滚动。点击已选中的按钮可收起面板。
2. **面板组件接口统一**：所有功能面板都移除自身的 `isOpen`/`onToggle` props，改为由父组件 `RightPanel` 控制渲染（条件渲染即可，无需 `isOpen`）。
3. **头像初版**：用 CSS 绘制的像素风占位 + Emoji 境界图标，后续再替换为真正的像素风图片资源。
4. **布局状态不持久化**：当前打开的面板（`activePanel`）不存入 `localStorage`，刷新后重置。

### 左栏（LeftPanel） — 200px 固定宽度

**上半部分：头像区域**
- 头像框：120×120px，居中，带像素风边框（金色/境界色渐变）
- 头像下方：道号（大字）+ 境界名（带境界色）
- 头像右上角小图标：可设置/换头像（后续功能）

**下半部分：核心属性**
- 纵向排列关键数据，每行一个属性 + 进度条：
  - ❤️ 体力 100/100
  - 🔮 灵力 20/20
  - ⚡ 精力 10/10
  - 📅 寿命 16.0/100
  - 💰 灵石 0
  - 修为进度条（带数值）
- 底部一个"📋 详细"按钮，点击后在中栏或浮层展示完整 StatusPanel

**样式要点：**
- 背景 `#12162a`，边框 `1px solid #222`
- 头像框使用 CSS border-image 或 box-shadow 实现像素风发光效果
- 固定高度跟随视口（`height: calc(100vh - 顶栏高度)`），内容超出时内部滚动

### 中栏（主区域） — flex:1

**操作按钮区：**
- 保持原有 4 个按钮（修炼/战斗/探索/休息），改为 2×2 网格排列
- 突破按钮条件出现时显示在按钮区下方，全宽

**日志区：**
- 占据中栏剩余空间
- 保持原有分类标签（全部/战斗/探索/奇遇/日常/系统）
- 日志列表区域高度自适应

### 右栏（RightPanel） — 280px 固定宽度

**按钮组区域（上部）：**
- 按分组排列，每组一个小标题 + 横排按钮
- 每个按钮：图标 + 文字，紧凑排列
- 选中按钮高亮（金色边框 + 背景加深）
- 按钮可带角标（如背包显示 `3/20`，商店显示灵石数）

**面板内容区域（下部）：**
- 点击按钮后，下方展开对应面板内容
- 面板内容区域可滚动
- 再次点击同一按钮可收起面板

**样式要点：**
- 按钮组和面板区域之间有明显分隔
- 面板区域背景 `#0e1020`
- 面板切换时可加简单的 fade/slide 过渡动画

### 头像系统

**初版方案（T0041 范围内）：**
- 使用 CSS 绘制的圆形/方形框 + 内部大 Emoji 作为占位头像
- 默认头像按性别区分：`🧑‍🦱` / `👩`（或更有修仙风格的 Emoji）
- 头像框边框色随境界变化：
  - 练气：`#9E9E9E`（灰）
  - 筑基：`#4CAF50`（绿）
  - 金丹：`#2196F3`（蓝）
  - 元婴：`#9C27B0`（紫）
  - 化神+：`#FFD700`（金）
- 创建角色时可选一个初始头像（从 3~5 个预设中选）

**后续扩展（不在本任务范围）：**
- 像素风头像图片资源
- 根据境界/装备自动改变头像外观
- 自定义头像上传

### 交互说明

1. **右栏面板切换**：点击按钮 → 下方展开对应面板；点击已激活按钮 → 收起面板；点击另一个按钮 → 切换面板
2. **左栏详细按钮**：点击"📋 详细"→ 在右栏切换到"状态详情"面板（复用 StatusPanel）
3. **突破按钮**：条件满足时在中栏操作区底部出现，保持呼吸灯动画效果
4. **头像点击（预留）**：后续可弹出头像选择/成就界面

### 响应式设计

| 断点 | 布局 | 说明 |
|------|------|------|
| ≥ 960px | 三栏（左200 + 中flex + 右280） | 完整桌面体验 |
| 768–959px | 两栏（左200 + 中flex），右栏变为底部抽屉或浮层 | 平板适配 |
| < 768px | 单栏，恢复类似当前的垂直布局 | 手机端保持可用 |

移动端适配策略：
- < 768px 时左栏收起为顶部横条（头像小图标 + 关键数值）
- 功能面板改为底部弹出式（从底部滑入）
- 操作按钮保持 2×2 布局
