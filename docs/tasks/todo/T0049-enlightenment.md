# T0049 — 悟道顿悟系统

- **状态**: ⬜ 未开始
- **分类**: 功法与技能
- **前置**: T0002, T0017
- **Spec**: —

## 描述

参考鬼谷八荒的悟道系统，为修炼增加"打坐悟道"玩法，在修炼过程中有概率进入悟道状态，触发顿悟获得功法、突破灵感等奖励。

## 核心需求

### 1. 悟道行为

- 在行动面板增加"悟道"按钮（与修炼并列）
- 悟道消耗时间（月），期间不能进行其他行动
- 悟道效果基于 `comprehension`（悟性）属性：
  - 低悟性：大概率无事发生，小概率获得少量修为
  - 高悟性：概率触发各种悟道事件

### 2. 悟道事件池

- **领悟功法**：从当前未学会的功法池中随机领悟一门（品质受悟性和境界影响）
- **技能升级**：已学技能熟练度大幅提升
- **突破灵感**：获得"灵感"buff，下次突破成功率 +X%（一次性消耗）
- **属性顿悟**：随机一项属性永久 +小幅提升
- **天道感悟**：获得特殊被动效果（如"天道感应"：暴击率 +5%，持续 X 回合）
- **走火入魔**（负面）：悟道失败，体力 -X%，或增加心魔值（与 T0050 联动）

### 3. 顿悟机制

- 修炼/战斗/探索等行为中有极小概率触发"顿悟"
- 顿悟不需要主动选择，自动触发
- 顿悟效果强于普通悟道（直接领悟高品质功法/获得大量修为/境界直接突破灵感）
- 顿悟触发记录在日志中，用金色高亮显示

### 4. 悟道环境加成

- 不同地图区域（T0021）提供不同的悟道加成：
  - 灵山：悟道概率 +20%
  - 瀑布：水系功法领悟率 +50%
  - 雷池：雷系功法领悟率 +50%
- 特殊物品可提升悟道效果（如"悟道茶"：本次悟道概率翻倍）

### 5. 悟道日志

- 悟道结果记入游戏日志
- 顿悟用特殊样式（金边/动画效果）展示

## 数据结构

interface EnlightenmentEvent {

}
