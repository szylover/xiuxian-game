# T0053 — 宗门管理（自建宗门）

- **状态**: ⬜ 未开始
- **分类**: 社交与NPC
- **前置**: T0027, T0025, T0021
- **Spec**: —

## 描述

参考鬼谷八荒后期玩法，在已有的门派系统（T0027，加入NPC门派）基础上，实现玩家自建宗门、收徒、管理、宗门争霸等后期玩法。

## 核心需求

### 1. 建宗条件

- 玩家达到一定境界（元婴期+）
- 拥有一块"灵脉宝地"（通过探索/购买获取，依赖 T0021 地图）
- 消耗大量灵石
- 必须离开当前门派（若已加入）

### 2. 宗门属性

```ts
interface SectState {
  id: string;
  name: string;          // 玩家自定义宗门名
  founderName: string;   // 创始人
  level: number;         // 宗门等级（1~10）
  reputation: number;    // 声望
  treasury: number;      // 宗门灵石库
  territory: string;     // 所在区域
  members: SectMember[]; // 门下弟子
  facilities: SectFacility[]; // 宗门设施
  techniques: string[];  // 宗门功法（可传授给弟子）
  alliances: string[];   // 同盟门派
  enemies: string[];     // 敌对门派
}
```

### 3. 收徒/招募

- 在地图各区域遇到散修 NPC 时可"招募入门"
- NPC 是否愿意加入取决于：宗门声望、宗主境界、宗门功法品质
- 弟子有资质（影响成长速度）和忠诚度
- 弟子上限受宗门等级限制（等级 1：5 人 → 等级 10：50 人）

### 4. 宗门设施

| 设施 | 功能 | 升级效果 |
|------|------|----------|
| 练功房 | 弟子修炼速度加成 | +10%/+20%/+30% |
| 藏经阁 | 存放功法，弟子自动学习 | 容量增加 |
| 丹房 | 宗门炼丹（弟子自动炼丹） | 炼丹成功率和产量提升 |
| 锻造室 | 宗门炼器 | 品质上限提升 |
| 灵田 | 自动产出灵草/材料 | 产量和品质提升 |
| 护山大阵 | 防御敌对势力入侵 | 防御力 |
| 宗门大殿 | 提升声望获取 | 弟子上限增加 |

- 设施建造和升级消耗灵石 + 材料 + 时间

### 5. 弟子管理

- 指派弟子任务：修炼/探索/护山/炼丹/外出历练
- 弟子自动成长（每月 tick 增加修为，尝试突破）
- 弟子可能叛逃（忠诚度过低）或陨落（外出历练危险）
- 杰出弟子可提拔为长老（管理层，分担管理负担）

### 6. 宗门争霸

- 声望排名：所有门派（包括NPC门派和玩家宗门）按声望排名
- 宗门战争：与敌对门派进入战争状态
  - 战争通过多场战斗（弟子 vs 弟子）结算
  - 胜利方获得对方部分资源和声望
  - 失败方声望大幅下降
- 宗门联盟：与友好门派结盟，战争时可请求援助

### 7. 宗门事件

- 随机宗门事件：
  - 弟子间争斗 → 需裁决
  - 外敌入侵 → 防御战
  - 天才弟子出现 → 特殊培养
  - 宗门秘境开启 → 团队副本机会
  - 其他门派来访 → 交流/结盟/挑衅

## 参考

- 宗门争霸是中后期核心玩法之一
