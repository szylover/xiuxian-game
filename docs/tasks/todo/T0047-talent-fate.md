# T0047 — 命格天赋系统

- **状态**: ⬜ 未开始
- **分类**: 核心循环
- **前置**: T0001
- **Spec**: —

## 描述

参考鬼谷八荒的命格系统，为角色提供开局天赋选择，影响整局游戏的修仙路线与属性成长。

## 核心需求

### 1. 天赋池

- 天赋分为多个类别：**体质类**（冰灵之体、雷灵根…）、**命格类**（天生道体、气运之子…）、**特殊类**（不死之身、万毒不侵…）
- 每个天赋定义：
  - `id`：`core:talent_ice_body`
  - `name`：冰灵之体
  - `quality`：品质（凡/良/优/极/仙，对应白/绿/蓝/紫/金）
  - `category`：体质 / 命格 / 特殊
  - `description`：详细说明
  - `effects`：效果列表（属性加成 / 特殊标记 / 解锁条件）
  - `exclusions`：互斥天赋列表（如冰体和火体不能共存）
- 天赋通过 DLC 注册（`DLCPack.talents?: TalentDef[]`）

### 2. 开局抽取

- 新建角色时展示天赋选择界面
- 随机从天赋池中抽取 3~5 个天赋供选择，品质受 `luck` 属性影响
- 玩家选择 1~2 个天赋（可选数量随难度设置变化）
- 高品质天赋出现概率低，仙品天赋 ≤1%

### 3. 天赋效果

- **属性加成类**：永久 +X% 某属性（如冰灵之体：灵力 +15%，火系抗性 +30%）
- **修炼加成类**：特定功法修炼速度 +X%，特定境界突破成功率 +X%
- **战斗特殊效果**：攻击附带冰冻/雷击概率，被攻击反弹 X% 伤害
- **解锁类**：解锁特定功法/神通的学习资格（如雷灵根才能学雷系神通）
- **经济类**：商店折扣、炼丹成功率加成、矿物获取双倍

### 4. 天赋面板

- 在状态面板中显示已拥有天赋（图标 + 名称 + 效果描述）
- 天赋按品质着色

### 5. 转世继承（与 T0030 联动）

- 转世重修时可保留部分天赋（高品质天赋保留概率更高）
- 转世次数越多，开局可选天赋数量/品质上限提升

## 数据结构

```ts
interface TalentDef {
  id: string;             // 'core:ice_body'
  name: string;           // '冰灵之体'
  quality: Quality;       // 'common' | 'good' | 'rare' | 'epic' | 'legendary'
  category: 'constitution' | 'fate' | 'special';
  description: string;
  effects: TalentEffect[];
  exclusions?: string[];  // 互斥天赋 ID
  weight: number;         // 抽取权重
}
```
