# T0055 — 天机榜（天骄排行）

- **状态**: ⬜ 未开始
- **分类**: 进阶机制
- **前置**: T0025, T0003, T0051
- **Spec**: —

## 描述

参考鬼谷八荒的天骄排行系统，实现一个全局修士排行榜，反映当前世界中所有修士（包括 NPC）的实力排名。排行榜不仅是展示，更是挑战目标——击败高排名修士可获取排名和奖励。

## 核心需求

### 1. 排行维度

| 排行榜 | 排序依据 | 说明 |
|--------|----------|------|
| 天机总榜 | 综合战力值 | 所有修士的总排名 |
| 境界榜 | 当前境界 + 修为 | 按境界排名 |
| 战力榜 | 战斗属性综合分 | 纯战斗力排名 |
| 炼丹榜 | 炼丹成功次数/品质 | 炼丹师排名 |
| 炼器榜 | 炼器成功次数/品质 | 炼器师排名 |
| 杀伐榜 | 击杀数 | PvP/PvE 击杀统计 |

### 2. 排行计算

- 综合战力 = f(境界, 攻击, 防御, 体力, 灵力, 功法品质, 装备品质, 天赋加成)
- 每月自动重新排名（NPC 实力变化后更新）
- 排名变动用箭头标记（↑ / ↓ / —）

### 3. 天骄称号

| 排名 | 称号 | 特效 |
|------|------|------|
| 第 1 名 | 天下第一 | 名字金色 + 光环 |
| 前 3 名 | 三大天骄 | 名字紫色 |
| 前 10 名 | 十大天骄 | 名字蓝色 |
| 前 50 名 | 天骄 | 名字绿色 |
| 其他 | 无称号 | 正常 |

- 获得天骄称号提升 NPC 对你的初始好感（"久仰大名"效果）
- 排名越高，被 NPC 主动挑战的概率越高

### 4. 挑战机制

- 玩家可主动挑战排名比自己高的修士（NPC）
- 挑战胜利：交换排名位置，获得奖励（灵石/声望/稀有物品）
- 挑战失败：无惩罚（仅消耗时间）
- 同一对手短期内不能重复挑战（冷却期 3 个月）
- NPC 也会主动挑战玩家（若 NPC 排名接近玩家且好战型）

### 5. 排行奖励

- 每年结算时，按排名发放奖励：
  - 第 1 名：仙品奖励
  - 前 3 名：极品奖励
  - 前 10 名：稀有奖励
  - 前 50 名：良品奖励
- 奖励通过事件系统发放（"天道馈赠"事件）

### 6. UI 设计

- 排行榜面板（可在右侧面板中打开）
- 展示：排名 / 名称 / 境界 / 战力 / 门派 / 排名变动箭头
- 玩家自己的排名高亮显示
- 点击 NPC 可查看详情 / 发起挑战
- 不同称号等级用对应颜色显示

## 参考

- 鬼谷八荒：天骄榜、地榜、天榜，击败排名靠前的人可获取其排名
- 排名是衡量玩家实力进度的核心指标，也是后期挑战目标
