# T0048 — 道侣双修系统

- **状态**: ⬜ 未开始
- **分类**: 社交与NPC
- **前置**: T0025, T0026
- **Spec**: —

## 描述

参考鬼谷八荒的道侣系统，实现 NPC 好感度 → 表白 → 结为道侣 → 双修加成的完整社交链路。

## 核心需求

### 1. 好感度系统（扩展 T0025 NPC 系统）

- 每个 NPC 有好感度值（0~100），分级：陌生（0~19）→ 认识（20~39）→ 友好（40~59）→ 亲密（60~79）→ 倾心（80~100）
- 好感度提升方式：
  - 赠送物品（不同 NPC 喜好不同，赠送喜欢的物品好感 +大，不喜欢 -小）
  - 完成 NPC 委托任务
  - 切磋/并肩作战
  - 日常互动（聊天/论道）
- `charisma` 属性影响好感度增长速率
- NPC 有性格标签（如：高冷、热情、傲娇、温柔），影响互动文本和好感增长曲线

### 2. 表白 / 结为道侣

- 好感度达到 80+ 后解锁"表白"选项
- 表白成功率受好感度、魅力、境界差距影响：
  - 境界差 ≥2 大境且对方更高：成功率大幅降低
  - 同境界或己方更高：正常
- 表白成功后双方结为道侣，状态面板显示道侣信息
- 同一时间只能有一个道侣（NTR-free）

### 3. 双修机制

- 道侣状态下解锁"双修"行为按钮
- 双修消耗时间（月），产出：
  - 双方均获得修为（量 > 单独修炼）
  - 概率触发"心意相通"特殊事件：临时属性大幅提升 / 领悟功法 / 突破灵感
  - 道侣亲密度（双修次数）影响加成系数
- 双修期间不能进行其他行动

### 4. 道侣战斗辅助

- 遭遇战斗时，若道侣在身边，自动加入战斗
- 道侣作为辅助单位：
  - 每回合有概率进行辅助攻击（伤害基于道侣属性）
  - 有概率施放辅助技能（加护盾/回血/增益 buff）
  - 不可被敌人直接击杀（退场机制：受到 X 次伤害后退出战斗）

### 5. 道侣事件

- 道侣特有随机事件：
  - 道侣被敌人掳走 → 救援任务
  - 道侣突破 → 协助渡劫事件
  - 道侣赠礼 → 随机获得稀有物品
  - 道侣吵架 → 好感度下降，需赠礼修复
- 道侣陨落 → 触发"心魔"事件（与 T0050 联动）

### 6. NPC 偏好数据

```ts
interface NPCPreference {
  likedItems: string[];      // 喜欢的物品 ID
  dislikedItems: string[];   // 讨厌的物品 ID
  likedTopics: string[];     // 喜欢的话题
  personality: 'cold' | 'warm' | 'tsundere' | 'gentle' | 'proud';
}
```
