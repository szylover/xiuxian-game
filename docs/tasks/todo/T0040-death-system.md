- **Spec**: [docs/specs/T0040-death-system.md](../specs/T0040-death-system.md)

## 描述

完整的死亡/后果/复活框架，替代当前仅有"寿元耗尽→游戏结束"的单一死法。

### 核心功能

1. **多种死亡触发源**：寿元耗尽、战斗阵亡、渡劫失败、心魔入体、气血耗尽、炼丹走火
2. **后果分级**：轻度（损失修为/灵石）、中度（降境界）、重度（游戏结束）
3. **护命机制**：护身灵符/玉碎护身/魂灯等道具可拦截致命伤害
4. **复活手段**：九转还魂丹、散仙化、转世重修等，通过 DLC 注册
5. **DLC 扩展**：所有死亡条件、护命道具、复活手段均通过 `registerDLC()` 注册
