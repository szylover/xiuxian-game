# T0050 — 心魔系统

- **状态**: ⬜ 未开始
- **分类**: 进阶机制
- **前置**: T0029
- **Spec**: —

## 描述

参考鬼谷八荒的心魔劫，为突破和修炼过程增加心魔挑战。心魔值随修炼积累，在关键节点触发心魔劫，必须化解才能继续前进。

## 核心需求

### 1. 心魔值

- 角色新增隐藏属性 `innerDemon`（心魔值，0~100）
- 心魔值积累来源：
  - 修炼（每次修炼 +微量，高境界积累更快）
  - 杀戮（每次战斗击杀 +少量）
  - 突破失败（+中量）
  - 使用邪功/禁术（+大量）
  - 道侣陨落/重大失败事件（+大量）
- 心魔值降低方式：
  - 悟道成功（-少量）
  - 使用"定心丹"等消耗品（-中量）
  - 佛门/儒道功法自带心魔抑制被动
  - 道侣双修（-少量）

### 2. 心魔劫触发

- 心魔值达到阈值（60/80/100）时，下次突破自动触发心魔劫
- 心魔值达到 100 时，即使不突破也会强制触发心魔劫
- 心魔劫与天劫不同：
  - 天劫 = 与天雷对抗（物理战斗）
  - 心魔劫 = 与内心幻象对抗（特殊回合制挑战）

### 3. 心魔劫战斗

- 心魔劫生成一个"心魔幻象"作为对手：
  - 外貌：玩家自身的镜像（相同的头像）
  - 属性：基于玩家当前属性 × 心魔值系数（心魔值越高，幻象越强）
  - 技能：复制玩家已学的功法/技能
- 心魔劫战斗规则：
  - 不能使用物品（纯靠实力）
  - 道侣不能辅助
  - 持续 N 回合，存活即为胜利（不需要击杀，只需撑过）
- 胜利奖励：心魔值清零，获得"心如止水"buff（修炼效率 +X%，持续一段时间）
- 失败惩罚：境界跌落一小级 / 体力大幅下降 / 走火入魔（随机属性永久 -X）

### 4. 心魔等级

| 心魔区间 | 等级 | 表现 |
|----------|------|------|
| 0~20 | 无心魔 | 正常状态 |
| 21~40 | 心绪不宁 | 修炼效率 -5% |
| 41~60 | 心魔初现 | 修炼效率 -10%，战斗偶尔失控（跳过回合） |
| 61~80 | 心魔缠身 | 修炼效率 -20%，突破成功率 -30% |
| 81~99 | 走火入魔 | 修炼效率 -40%，随机掉属性 |
| 100 | 心魔爆发 | 强制触发心魔劫 |

### 5. UI 展示

- 状态面板显示心魔值（用暗红色进度条）
- 心魔等级用文字标注（无心魔 / 心绪不宁 / …）
- 心魔劫战斗弹窗用特殊暗色主题区别于普通战斗

## 参考

- 鬼谷八荒：突破时触发心魔劫，心魔是自身镜像，属性更强
- 心魔劫难度随境界提升而增加
