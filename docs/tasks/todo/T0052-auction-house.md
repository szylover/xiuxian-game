# T0052 — 拍卖行

- **状态**: ⬜ 未开始
- **分类**: 物品与经济
- **前置**: T0025, T0015
- **Spec**: —

## 描述

参考鬼谷八荒的拍卖会，实现定期举办的拍卖行系统，NPC 竞价，玩家可参与竞拍珍稀物品。

## 核心需求

### 1. 拍卖会触发

- 拍卖会定期举办（每 N 个月/每年一次，可配置）
- 由特定城市/门派举办，需要玩家前往对应区域（依赖 T0021 地图）
- 不在场时收到"拍卖会即将开始"的通知事件

### 2. 拍品生成

- 拍品从物品/装备/丹药/功法的全局注册表中随机抽取
- 拍品数量：每次拍卖 5~10 件
- 拍品品质与举办方势力挂钩：
  - 小城坊市：普通~良品
  - 大宗门拍卖：良品~极品
  - 天下拍卖会（稀有事件）：极品~仙品
- 压轴拍品（最后一件）品质保底更高

### 3. 竞拍流程

- 每件拍品展示：名称、品质、属性/效果、起拍价
- 竞拍进行 3~5 轮：
  - 每轮 NPC 根据自身财力和需求决定是否加价
  - 玩家可选择出价/放弃
  - 出价必须 > 当前最高价（加价幅度有最低限制）
- NPC 竞拍行为：
  - 富有 NPC 更愿意高价竞争
  - NPC 对自己需要的物品（如缺装备的 NPC）出价更积极
  - NPC 有心理价位上限，超过后自动放弃

### 4. 拍卖结束

- 最高出价者获得拍品
- 灵石从获得者扣除
- 若玩家赢得拍品，物品直接进入背包
- 若无人出价，拍品流拍

### 5. 玩家寄卖

- 玩家可将自己的物品寄卖到拍卖行
- 设置底价，拍卖行收取手续费（成交价的 5~10%）
- 若未成交，物品退回背包

### 6. UI 设计

- 拍卖行弹窗/面板：
  - 当前拍品展示（大图 + 属性详情）
  - 当前出价 + 出价者名称
  - "出价"按钮 + 自定义出价金额
  - 竞拍历史（谁出了多少）
  - 拍品列表（可预览后续拍品）

## 参考

- 鬼谷八荒：定期拍卖会，NPC 竞价，可买到稀有功法/法宝
- 拍卖是获取高品质物品的重要渠道
